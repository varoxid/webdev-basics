# Multipaint

## Description

MultiPaint is a multi-user web application for real-time collaborative drawing.

### Key features:

- Co-drawing with other users
- Custom drawing settings
- Selecting design themes

### Technologies:

- HTML5
- Canvas API
- WebSockets
- WebStorage
- File API
- Geolocation API

## Set up

```
npm install
```

```
npm start
```

Open in browser:

```
http://localhost:8888
```
